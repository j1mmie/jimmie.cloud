Personal website of Jimmie Tyrrell

Built with Bun, Typescript, React, Vite, and Vercel

[https://jimmie.cloud](https://jimmie.cloud)

Setup:
1. Install mise: `brew install mise`
2. Run `mise install` to install dependencies (bun)
3. Run `bun install` to install packages
4. Run `bun dev` to start the development server
5. Run `bun build` to create a production build

Deploying:
1. Push to get - should auto deploy via Vercel
